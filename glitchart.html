The tools utilized to make this project were Adobe Photoshop, Audacity and TextEdit.
This series provides some insight into the misogny as well as lack of POC representation
in the Twitch community and platform. Included are glitched images of two very
well-known female streamers, as well as the Twitch logo itself. In this process, I
used data blending and data moshing, which are methods that utilize TextEdit and Audacity
to change the code or soundwave from the original images to make a glitch effect on these images.
